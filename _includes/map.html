{% comment %} Renders a MapLibre GL map of pinned places. Usage: {% include map.html places=site.locations %} Parameters: places (required) a list of location documents (site.locations). Each document's front matter provides: lat, lng numeric coordinates title pin title, shown bold atop the popup type optional marker style key links optional list of { text, url } hashes The document body becomes the place page; its excerpt (first paragraph) is shown in the popup. Any blog post whose front matter sets `location:` to a document's slug is linked from that pin automatically. id (optional) DOM id for the map container; defaults to "map". Pass a distinct id when placing more than one map on a page. only (optional) a single location slug. When set, only that pin is rendered, the map is centered on it at a fixed zoom rather than framed to bounds, and no popup is attached (the host page shows the place's details itself). The host page must set `map: true` in its front matter so the layout loads the MapLibre GL assets. -- claude, 2026-07-03 {% endcomment %} {% assign map_id = include.id | default: "map" %}
